# Processamento de imagem por meio de padrões RGB em C - Estrutura de Dados I

Desenvolvimento de um software que realiza o processamento de imagens por meio de padrões RGB, utilizando a linguagem de programação C. O projeto foi desenvolvido para a disciplina de Estrutura de Dados I, do curso de Sistemas de Informação da Universidade Federal do Piauí (UFPI).

## Começando

Essas instruções fornecerão uma cópia do projeto em execução na sua máquina local para fins de teste. 

**Versão do GCC**: `8.1.0`


**Sistema operacional utilizado:** `Windows`

### Configurando o ambiente (No Windows)

1. **Comando 1**: 
```bash
gcc -c image.c -o image.o
```

2. **Comando 2**:

```bash
gcc -c main.c -o main.o
```
3. **Comando 3**: Aqui você irá executar o arquivo que você criou.
```bash
gcc image.o main.o -o main
```
4.  **Rodar Executavel gerado:**: 
```bash
./main
```
